Ribs.support.mixins.renderChain

Creates a more sophisticated component life cycle inside the render function.
Defined in: renderchain.js.

Properties
inheritingMethods
Default Value:
["unbindEvents", "bindEvents", "redraw", "refresh", "hide", "dispose"]
initialized
This flag should be set to true externally when the component has finished initializing.
Default Value:
false
invalidated
When calling render(), if this flag is set, a redraw will occur. When redraw is finished, this flag is again reset to false.
Default Value:
true
Methods
bindEvents()
Set all listeners to DOM events. This is an empty function by default.
dispose()
Destroy the View beyond restoring.
hide()
Remove from DOM with the prospect of reattaching.
redraw()
Do all expensive DOM operations that could break listeners (hiding etc.). This is an empty function by default.
refresh()
Set styles, calculates values, etc. This is called each time the View is rendered, so don't do anything expensive here. This is an empty function by default.
render()
Override to Backbone.View.render.

Exits if initialized flag is not set.

Maintains the View life cycle of rendering. If invalidated flag is set, unbinds all DOM events, calls redraw, and then binds the events. After this comes always refresh, regardless of the invalidated flag.
unbindEvents()
Clear all listeners to DOM events. This is an empty function by default.

Ribs.support.mixins.templated


Defined in: templated.js.

Properties
json
"Data" to give to the template function. Should be calculated right before rendering the template in redraw.
overwriteEl
If set to true, will replace the el property with the template element. Otherwise replaces the contents of the el with the templated element, leaving the root tag.

Notice that you will have to re-append this el to DOM manually in case it is overwritten.

Default Value:
false
templateFunction
Template function to call when using the template. Gets the model json as its first parameter.

Overrides templateSelector and templateString
Default Value:
null
templateSelector
jQuery selector for the template element.
Default Value:
null
templateString
Plain string to use as the template.

Overrides templateSelector.
Default Value:
null
Methods
mixinInitialize()
Initializes the template based on the mentioned fields.
redraw()
This is where the template is applied.
